# Pac-Man 3D

Pac-Man remake using Unity and C#\
First-person view with Horror Assets\
Implements Ghost Pathing using Raycasting

###### Program Keybinds

W = Move Forward\
A = Move Left\
S = Move Back\
D = Move Right\
Use Mouse to look around
